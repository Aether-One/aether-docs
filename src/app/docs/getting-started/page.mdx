# Installation

Aether is distributed as a **standalone binary** — no Node.js or any runtime required.
Just download, make it executable, and run. The commands below always fetch the **latest release** automatically.

## Install via CLI

Choose your platform and run the commands in your terminal:

---

## Verify Installation

```bash
aether --version
```

## Quick Start

Just run `aether` in any project directory:

```bash
cd my-project
 aether
```

An interactive chat opens. Type `/help` to see available commands.
