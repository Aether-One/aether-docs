# Installation

## Requirements

- **Node.js** 20+
- **OS**: Windows, macOS, or Linux

## Install via npm

```bash
npm install -g aether
```

## Or run directly (no install)

```bash
npx aether
```

## Development (from source)

```bash
git clone https://github.com/aether-one/aether
 cd aether
 npm install
 npm run dev
```

## Verify

```bash
aether --version
```

## Usage

Just run `aether` in any project directory:

```bash
cd my-project
aether
```

An interactive chat opens. Type `/help` to see available commands.

## Available Commands

| Command | Description |
|---------|-------------|
| `/help` | Show available commands |
| `/genesis` | Analyze and document your project |
| `/exit` | Exit Aether |
| `/clear` | Clear the screen |

## Features

- **Tab completion** — Tab to accept suggestions
- **Tips** — Helpful hints shown periodically
