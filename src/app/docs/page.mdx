# Welcome to Aether

Aether is an open-source CLI that transforms any codebase into an AI-native workspace. Think of it as Genesis — the birth of intelligence within your project, like a galaxy forming from stardust.

## Why Aether?

Modern development demands more than just a code editor. You need a tool that **understands** your codebase — its structure, patterns, dependencies, and conventions.

Aether bridges the gap between your code and AI, giving you:

- **Instant context** — No more copy-pasting files into chat windows
- **Privacy first** — Your code stays on your machine
- **Provider agnostic** — Use any AI model you prefer
- **Zero config** — Just run `aether genesis` and you're ready

## How It Works

```bash
# Install
curl -fsSL https://aether.sh/install | sh

# Initialize in any project
cd your-project
aether genesis

# Start asking questions
aether ask "explain the authentication flow"
```

## Philosophy

The naming is intentional. In the Aether universe:

- **Genesis** — The birth of your AI workspace, like a galaxy forming
- **Context** — The nebula of knowledge surrounding your code
- **Providers** — The stars that power your queries (OpenAI, Anthropic, local models)
