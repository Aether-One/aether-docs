# Welcome to Aether

Aether is an open-source interactive CLI that transforms any codebase into an AI-native workspace.

## Why Aether?

Modern development demands more than just a code editor. You need a tool that **understands** your codebase — its structure, patterns, dependencies, and conventions.

Aether bridges the gap between your code and AI, giving you:

- **Instant context** — Understands your codebase structure, patterns, and conventions
- **Privacy first** — Your code stays on your machine
- **Provider agnostic** — Supports OpenAI, Anthropic

## How It Works

```bash
# Install
npm install -g aether

# Run in any project
cd your-project
aether
```

Aether opens an interactive chat. Use `/<COMMAND>`.

## Philosophy

The naming is intentional. In the Aether universe:

- **Genesis** — The birth of your AI workspace
- **Context** — The knowledge surrounding your code
- **Providers** — The models that power analysis (OpenAI, Anthropic, Ollama)
