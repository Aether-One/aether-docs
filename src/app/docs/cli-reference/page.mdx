# CLI Reference

## Usage

```bash
aether [flags]
```

Aether opens an interactive chat interface. No subcommands needed.

## Flags

| Flag | Description |
|------|-------------|
| `--no-animation` | Skip startup animation |
| `--version`, `-v` | Show version |

## Interactive Commands

Once inside the chat, use slash commands:

| Command | Description |
|---------|-------------|
| `/help` | Show available commands |
| `/genesis` | Analyze and document your project |
| `/exit` | Exit Aether |
| `/clear` | Clear the screen |

## Dropdown Navigation

Type `/` to see all commands. Use ↑↓ arrows to navigate, Enter to select.
